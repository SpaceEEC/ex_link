    ExLink v0.1.0 API Reference

    Modules

    ExLink

    Handles WebSocket connections to a Lavalink node, providing:

    • ExLink.Players to keep track of Lavalink players and control them
    • ExLink.Messages to send via ExLink.Connection(s)
    ExLink.Connection

    Handles the connection to a Lavalink node

    ExLink.Message

    Module used to build Outgoing Messages to send to a Lavalink node

    ExLink.Player

    Player behaviour used to keep track of running players and manage them
